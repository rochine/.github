# Contributing to Rochine Labs

We welcome contributors passionate about robotics, Solana, and verifiable automation.

## Setup
1. Fork repo
2. Work in a feature branch
3. Submit PR with clear title & description

## Commit Style
Use concise, meaningful commits:
- `feat:` new feature  
- `fix:` bug fix  
- `docs:` documentation  
- `refactor:` code cleanup

## Development Stack
- Solana Anchor (Rust)
- Next.js + Tailwind (Console)
- Node/Python SDK for RobotLink
