# Security Policy

## Supported Versions
Beta builds are continuously updated on DevNet.

| Version | Supported |
|----------|------------|
| Beta (current) | ✅ Yes |
| Alpha (archived) | ❌ No |

## Reporting a Vulnerability
Please email [security@rochine.tech](mailto:security@rochine.tech)  
We aim to respond within **48 hours** for any reported vulnerability.
